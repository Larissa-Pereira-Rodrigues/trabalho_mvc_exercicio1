# Sistema de Aluguel de Quadra

## Classes

- Cliente
- Horario
- Aluguel

## Regras

- Não pode cadastrar cliente sem nome
- Não pode cadastrar horário com valor negativo
- Não pode reservar um horário já ocupado
- Um cliente pode reservar mais de um horário
- O sistema calcula o valor total automaticamente
- Deve mostrar os alugueis feitos no dia

## Relações

- Um cliente pode ter vários alugueis
- Um horário pode estar em apenas um aluguel por vez
